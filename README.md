# Transfer Learning for image classification using [PyTorch](https://pytorch.org/get-started/locally/)
This is a project code for a transfer learning project. In this project, image classifier is built with [Pytorch](https://pytorch.org/get-started/locally/) by applying the concept of transfer learning.

## Installation Requirements
- [Python](https://www.python.org/downloads/)
- [PyTorch](https://pytorch.org/get-started/locally/)
- [Pillow](https://pillow.readthedocs.io/en/stable/installation.html)
- [NumPy](https://numpy.org/)
- [matplotlib](https://matplotlib.org/)
